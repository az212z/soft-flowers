# DESIGN-QUALITY-REPORT — الأزهار الناعمة (Soft Flowers)

## 1) المهارات المُستدعاة وكيف طُبِّقت

### `ui-ux-pro-max` (+ `--design-system`)
شُغّل: `search.py "florist flower shop soft pastel airy" --design-system`.
- **المخرجات:** نمط *Hero-Centric + Conversion*؛ ستايل *Organic Biophilic* (طبيعي، أشكال عضوية، أخضر/وردي)؛ ألوان أساسها أخضر طبيعي + وردي زهري؛ تأثيرات زوايا دائرية 16–24px وظلال طبيعية.
- **التبنّي:** أخذنا روح *Organic Biophilic* (نعناع + خوخي + زوايا ناعمة 12–28px + ظلال خفيفة) لكن **خفّفنا التشبّع** نحو باستيل هوائي بدل الأخضر الغامق ليطابق بريف "soft airy pastel" ويتميّز عن المواقع الشقيقة. الخط المقترح (Varela Round) رُكني الإحساس؛ نفّذناه عربيًا عبر **El Messiri + Tajawal** (نظير ناعم/ودود). طبّقنا قائمة ما-قبل-التسليم: لا إيموجي كأيقونات (SVG inline)، حالات hover 150–300ms، تباين ≥4.5:1، focus ظاهر، احترام reduced-motion، breakpoints 375/768/1024/1440.

### `design-taste-frontend` (anti-slop)
- **Design Read:** صفحة هبوط لمحل ورد، جمهور مُهدون/مناسبات في أبها، لغة باستيل-هوائية، Dials: VARIANCE 7 / MOTION 6 / DENSITY 3.
- **منع التكرار:** هيرو غير متمركز (split غير متماثل 1.05fr/0.95fr)؛ تنويع أنظمة التخطيط (هيرو split، trust 4-chips، خدمات 4-grid، why media+قائمة مرقّمة، gallery بنتو مع خلية wide، order نموذج+aside، location، CTA متمركز) — ≥6 عائلات تخطيط.
- **eyebrow restraint:** الـ eyebrow بنمط ناعم (نص خوخي + شرطة) لا uppercase tracking مكرّر.
- **ممنوعات احترمناها:** صفر em-dash (`—`)؛ لا أسعار مخترعة ("حسب الطلب")؛ صور حقيقية فقط لا fake-screenshots؛ accent واحد مقفل (خوخي-طيني) عبر الصفحة؛ نظام زوايا متّسق؛ ظلال مائلة للخوخي لا أسود صريح؛ CTA واحد لكل نيّة.

### `emil-design-eng` (الموشن والتفاصيل)
- منحنى `--ease: cubic-bezier(0.23,1,0.32,1)` (ease-out قوي) للدخول.
- أزرار: `:active { scale(.97) }` رد فعل لمسي فوري؛ مدد قصيرة (160ms للضغط، 250–600ms للريفيل).
- لا حركة على أفعال متكرّرة؛ الموشن مُبرَّر (بتلات = مزاج الورد، bloom = تفتّح، ken-burns = حياة، sheen = دعوة للضغط).
- البتلات تبدأ من opacity 0 وتتفّتح الكروت من `scale(.92)` لا من صفر؛ stagger 40ms.
- transform/opacity فقط (GPU)؛ reduced-motion يوقف البتلات والكين-بيرنز والتحوّلات.

## 3) قرارات UI/UX الأساسية
- **قائمة جوال ملء الشاشة** (100vw/100dvh، خلفية صلبة، زر X واضح، `role=dialog` + `aria-modal`)، تُغلق بـ Escape والروابط.
- **نموذج طلب**: labels فوق الحقول، أخطاء أسفل الحقل، تحقّق جوال سعودي، يبني `wa.me` + يحفظ localStorage + toast (`aria-live=polite`).
- **gallery lightbox** خفيف بإدارة focus وإرجاعه.
- **FABs** واتساب/اتصال/خرائط ثابتة.
- صور بأبعاد محدّدة (منع CLS) + lazy لغير الهيرو + `fetchpriority=high` للهيرو.

## 4) سبب الألوان والخطوط
الخوخي + النعناع + الأبيض = إحساس ورد ناعم وأنثوي-محايد ومناسب لمناسبات الإهداء؛ الرمادي الدافئ يريح العين ويبقي التركيز على الصور. El Messiri يعطي رُقيًّا عربيًّا للعناوين، وTajawal نظيف وعالي القراءة للنص. خفّفنا تشبّع توصية ui-ux-pro-max لنحقّق "airy pastel" المطلوب والتمايز عن المواقع الشقيقة.

## 5) إمكانية الوصول (Accessibility)
- **تباين:** كل أزواج النص/الخلفية ≥4.5:1 (مُحقّقة حسابيًّا؛ الـ accent عُدِّل من #C9745F إلى #A24E3B لاجتياز AA على كل الغسلات).
- HTML سيمانتيك (header/nav/main/section/footer)، تدرّج h1→h3 بلا قفز.
- كل صورة لها alt عربي؛ كل زر أيقونة له aria-label.
- `:focus-visible` بـ outline نعناعي 3px على كل تفاعل.
- اللون ليس وحده ناقلًا للمعنى (أيقونات + نص)؛ دعم كامل لـ `prefers-reduced-motion`.
- أهداف لمس ≥48px (أزرار/برغر/FAB/حقول).

## 6) الحركة (Motion)
بتلات منجرفة (9، أحجام/سرعات عشوائية)، تفتّح كروت bloom (scale .92→1 + fade + stagger 40ms عبر IntersectionObserver مع fallback بعد 2.5s)، ken-burns 22s، تكبير صور 1.03–1.05 عند المرور، sheen على الـ CTA. عنصر/عنصران متحرّكان كحد أقصى لكل مشهد. كله transform/opacity ويُوقَف عند reduced-motion.

## 7) اختبار الذوق (Taste / Impeccable)
هل يبدو فاخرًا؟ نعم — مسافات سخيّة، ظلال مائلة للخوخي، صور كبيرة. سعوديًّا مناسبًا؟ نعم — RTL، نص محايد جندريًّا، تقييم قوقل. يقنع خلال 3 ثوانٍ؟ الهيرو يوصل القيمة + CTA + دليل (4.8/446) فورًا. لا يشبه قالبًا مجانيًّا؟ هوية باستيل مخصّصة وموشن وردي بصري. ✅

## 8) النصوص
عربية سعودية طبيعية، **محايدة جندريًّا** (اطلب/تواصل/شاهد — لا اطلبي/لكِ)، بلا وعود مبالغ، بلا أسعار مخترعة. التقييم حقيقي من قوقل.
